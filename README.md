# 微信开发入门 -- 个人学习总结

- 附加：本微信学习使用YouMeek的SSM脚手架，YouMeek博客地址：http://www.youmeek.com/

    - IDE：IntelliJ IDEA 15.0.4
    - SSM 版本：
        - spring version：**3.2.16.RELEASE（也支持 4.1、4.2）**
        - mybatis version：**3.3.1**
        - mybatis-spring version：**1.2.4**
